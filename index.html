zls in the browser

Some quick notes:

  • The editor might take a bit to load (1-3 seconds)
  • Opening devtools in Chrome causes a stack overflow that otherwise doesn't occur :/
  • See the source and learn more @ https://github.com/zigtools/zls-in-the-browser

This demo supports completions, formatting (Ctrl+S), and semantic token-based highlighting all thanks to a wasm32-wasi build of zls!
